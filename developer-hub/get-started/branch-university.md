## What is Branch University?

Branch University is a self-paced learning platform for mastering deep linking and attribution with Branch. Through hands-on modules, step-by-step demos, and on-demand video tutorials, you'll learn how to create, test, troubleshoot, and report on deep links across all your marketing channels.

Key features include:

- **Learning paths**: Curated paths covering common mobile marketing tasks — from deep linking basics to ad attribution.
- **Training modules**: In-depth modules with realistic scenarios and best practices for social post deep links, display ad attribution, mobile web banners, and more. Each module includes step-by-step instructions for creating, testing, and troubleshooting deep links.
- **Certifications**: Validate your expertise with Branch certifications. Earn the Branch Mobile Growth Certified badge, with more advanced certifications coming soon.

## Access Branch University

You need a Branch account to use Branch University. Select **Login** on the Branch University site, and you'll be prompted to enter your Branch credentials.

::: info Note
If you log in to Branch through an SSO provider like Okta or OneLogin, make sure you're already logged in before navigating to Branch University.
:::

## Find Branch University

You can get to Branch University three ways:

- **Go directly** — Visit <https://university.branch.io/>.

- **From Branch** — Click the help icon (?) in Branch and select Branch University from the drop-down menu.

- **From the Help Center** — Look for University links on the Help Center home page and in the site header.
