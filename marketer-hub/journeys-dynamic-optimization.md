Use Journeys Dynamic Optimization to automate creative testing and visitor allocation percentage adjustment.

## Set up Journeys Dynamic Optimization

### Before You Begin

[Create a Journey](create-journey-banner-or-interstitial.md) if you don’t already have one that you want to optimize. Make sure your Journey has at least two creatives before you set up Journeys Dynamic Optimization.

::: info Note
Make all the changes that you want to make to your Journey before setting up Dynamic Optimization. If you change anything after completing these steps, it will end the automated testing and optimization.
:::

### Step 1: Choose a Campaign Optimization Metric

Choose a campaign optimization metric to allow targeted performance tracking and optimization.

You can choose one of the following metrics:

- **Acquisition**: The number of unique users who installed your app from the selected Journey.
- **Engagement**: The number of unique users who opened your app from the selected Journey.

Complete the following steps to select a campaign optimization metric for a Journey:

1. In the Branch Dashboard, select the **Journeys** tab. The **Journeys** page will open.
2. On the **Journeys** page, select the **Manager** tab.
3. Select the three-dot icon on the right side of the Journey you want to change.Select **Edit**. The **Select Audience** page will open.
4. Under **Select Campaign Objective**, select the checkbox next to **Acquisition** or **Engagement**. **Acquisition** is selected by default.
5. Select the **Continue** button. The **Configure Creatives** page will open.

### Step 2: Enable Dynamic A/B Testing

Enable dynamic A/B testing for a Journey’s creatives.

::: info Note
To use dynamic A/B testing, your selected Journey must have at least two creatives.
:::

Complete the following steps to enable dynamic A/B testing for a Journey:

1. On the **Configure Creatives** page, select the toggle next to **Enable Dynamic A/B Testing**.
2. Select **Continue**. The **Validate & Test** page will open.
3. Select **Start Journey**.
