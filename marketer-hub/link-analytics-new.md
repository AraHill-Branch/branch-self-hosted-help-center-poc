## Introduction

The Link Analytics data source provides link-level reporting on Branch link performance. Use it to understand how your links are performing across channels, campaigns, and platforms.

To learn more about Analysis, read our [Dashboards Overview](analysis-dashboards-overview.md) article. To learn about all available data sources, read our [Analysis Data Sources](analysis-data-sources-new.md) article.

## When to use this data source

Use the Link Analytics data source when you want to analyze link-level performance data, including clicks, impressions, and downstream conversions attributed to specific Branch links. This data source includes link-specific dimensions like Link Id, Link Title, Domain, and Browser that are not available in the standard Activity data source.

::: tip Tip
When [creating a widget](analysis-dashboards.md#create-and-configure-widget), select **Link Analytics** to access these fields.
:::
