::: info Note
This article describes the legacy Branch method for adding additional apps. For new Branch information, check out our [New Branch articles](https://help.branch.io/docs/new-branch).
:::

1. Locate your app name in the top left corner of Branch, and select the drop-down arrow to open the menu.
2. At the bottom of this menu, select **Create New App**.
3. In the **Create New App** pop-up modal:  
    a. Provide the **App Name**.  
    b. Select the correct **Timezone**.
4. Select **Create**.
