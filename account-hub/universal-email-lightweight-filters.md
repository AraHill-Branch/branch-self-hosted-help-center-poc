Branch Email analytics contains new improvements to filtering. Check out what's new here 📧

## Overview

Last year, we updated Branch analytics reports for Ads, SEO App Attribution, and OTT with more unified and modern reporting capabilities. Branch Email is now adopting some of these upgrades! When viewing your Branch Email analytics, you'll now be greeted with a new lightweight filter with the following features:

- Updated modern style
- Stays sticky as you scroll
- Filters that apply to both the Trends chart and Events table
- Better saved views switcher
