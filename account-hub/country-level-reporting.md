## Overview

You can now report on your Google Ads and TikTok campaign performance by **Country**. In addition to **Country**-level reporting, we will now populate the **Channel** and **Secondary Publisher** dimensions with descriptive values to help you understand more about your Google Ads campaign performance.

## View Country-Level Metrics

Complete the following steps to view **Country**-level metrics for Google Ads and TikTok:

1. In the Branch Dashboard, select **Ads** > **Analytics** > **Activity**.
2. In the **Events** section, enter “country” in the **Compare by** text field.
3. Select “Enter” or “Return” on your keyboard to add the dimension.

You can now view or export your metrics like ROAS, Cost, Clicks, Impressions, attributed events, and more sorted by **Country**.

::: warning Note
The **OS** and **Platform** dimensions can’t be combined with **Country** in Google Ads or TikTok reports. We’ll provide more compatibility information in the near future.
:::

## View Channel and Secondary Publisher Metrics

Complete the following steps to view **Channel**- or **Secondary Publisher**-level metrics for Google Ads:

1. In the Branch Dashboard, select **Ads** > **Analytics** > **Activity**.
2. In the **Events** section, enter “channel” and/or “secondary publisher” in the **Compare by** text field.
3. Select “Enter” or “Return” on your keyboard to add the dimension.

You can now view or export your metrics like ROAS, Cost, Clicks, Impressions, attributed events, and more sorted by **Channel** or **Secondary Publisher**.

### Expected Values

You can expect the following values to populate your **Channel** field:

- Display
- GoogleTV
- Cross-network
- Search
- Youtube
- Unpopulated

You can expect the the following values to populate your **Secondary Publisher** field:

- Display
- GoogleTV
- Cross-network
- GoogleSearch
- SearchPartners
- YouTubeVideos
- Unpopulated

::: info Note
We surface as much insight as possible based on the data we receive from Google. If we don’t receive values that allow us to report with confidence, the data will be marked “Unpopulated.” We will enhance these available values as more data becomes available.
:::
