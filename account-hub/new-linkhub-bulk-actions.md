We are excited to announce some new LinkHub functionality: bulk actions!

## Overview

[LinkHub](linkhub.md) lets you manage links at scale and access link insights through enhanced reporting. New bulk action features are now available for LinkHub. Along with the already existing *Collection* functionality, you'll now be able to do the following in bulk:

- Add/Edit Links
- Copy
- CSV Export
- Create QR Codes
- Archive
- Delete
