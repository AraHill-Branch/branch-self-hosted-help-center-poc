We have implemented significant enhancements to UI components of the Branch Dashboard. View them here! 🎇

## Overview

The Branch Dashboard received a fresh coat of paint to help make creating links and analyzing marketing performance a much more pleasant experience. The following enhancements have been implemented:

- Enhanced font size
- Enhanced letter spacing
- Enhanced font-weight
- Improved spacing
- Improved Card border-radius
- Updated form field styles
- Updated colors to enhance accessibility (aligned with other Branch marketing pages)
- Updated old competent themes (colors & typography).

::: tip
Ready to play around? Visit the Branch Dashboard here: <https://dashboard.branch.io/>
:::
