## Overview

Advanced Compliance is a **premium** solution from Branch that offers enhanced privacy and data security measures, enabling customers in highly regulated industries to use Branch's linking and measurement services while helping them to fulfill their heightened compliance requirements. Some of the advanced security configuration features include:

- Advanced Security Gateway - A gated data architecture, that includes an isolated network with custom traffic routing, more robust authentication, and additional levels of encryption.
- Protected API - Data will flow through a gated and segregated environment, in which access to customer data is restricted based on business needs.
