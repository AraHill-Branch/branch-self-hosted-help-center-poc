## Overview

The **Recent Activity** tab allows you to view changes made in your Branch account within the last 90 days and provides the following benefits:

- **Accountability** - View logs of users' actions.
- **History** - See edit history for all products, configuration, and more, based on your account timezone.
- **Accessibility** - Available to users with **Read** access to **Sensitive Data** and also **Admins**.

## Get started

To access the Recent Activity page, navigate to **Configuration > Testing & Monitoring** and select the **Recent Activity** tab.

You can filter the activity log by using the search bar or by selecting a specific feature from the **All features** dropdown.

## Supported features

- **Quick Links (Short Links)**

  - New Short Link created.
  - Existing Link edited.
  - Existing Link deleted.
  - Existing Link archived.
- **Ads**

  - **Partner Management**

    - Partner Enabled/Disabled.
    - Partner Settings.
    - Postbacks - Edit, enable/disable, delete.
    - Attribution Windows.
  - **Ad Links.**
- **Account Settings.**
- **Journeys**

  - Created.
  - Edited.
  - Changes in Status (Active, Stopped, Archived).
  - Changes in Priority.
  - Changes in Schedule.
- **Configuration**

  - General URI Scheme Mode.
  - Link Settings.
- **Email**

  - Partner Enabled/Disabled.
- **Data Feeds**

  - **Data Integrations**

    - Partner Enabled/Disabled.
  - **Webhooks**

    - webhook Enabled/Deleted/Archived.
  - **Settings.**
- **SKAdNetwork**

  - Created/Updated/Deleted Conversion Value.
