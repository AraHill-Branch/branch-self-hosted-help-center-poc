The Help Center received a fresh coat of paint! The redesign brings along many improvements to the user experience of the documentation 🎨

## Overview

The Help Center needed a complete redesign. These improvements include:

- Matching other Branch web properties
- Better support for small screens or mobile devices
- Improved Search UI
- Improved readability in dark mode
