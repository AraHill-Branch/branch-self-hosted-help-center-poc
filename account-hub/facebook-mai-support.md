Branch now supports Ads customers who are leveraging Meta's AEM solution for **Mobile App Install** in conjunction with the previously released Mobile App Engagement 🙌

## Overview

As a reminder, we're making it easier than before for our valued customers to expand their reach and enhance their ad measurement experience for iOS 14.5+ app campaigns, now for user acquisition objectives. [Aggregated Event Measurement (AEM)](https://www.facebook.com/business/help/721422165168355?id=1877298665783613) is Meta's solution to help advertisers reach more people and measure ad performance on their iOS 14.5+ App Campaigns.

While there are many different campaign targets the overarching objectives are as follows:

| Campaign Type | Description |
| --- | --- |
| Mobile App Engagement | Aggregate Event Measurement is currently available for the following types of campaigns when the destination is set to a mobile app: sales campaigns with catalog turned on or off, lead campaigns, and engagement campaigns. |
| Mobile App Install | Aggregated Event Measurement has expanded to include campaigns that use the app promotion objective. This means you can now run app promotion campaigns without configuring app events for Apple's SKAdNetwork, and more app events will be available for ad delivery optimization. You can continue to configure events for SKAdNetwork if you want to view performance across platforms. |

::: info
Re-toggling

You may need to re-toggle the **Enabled Aggregated Event Measurement (AEM)** toggle in the Branch Dashboard if the MAI supported SDK version is updated or if AEM was enabled before MAI rediness release.
:::
