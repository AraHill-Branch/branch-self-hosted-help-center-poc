We've added local currency support for Unified Analytics. Learn more here 📊

## Overview

Starting today, you can select your local currency when viewing your Unified Analytics reports! This means that revenue-related columns like revenue, cost, ECPC, ECPI, etc. can now be displayed in your local currency.

![](/img/643cc0e-Unified_Local_Currency.png)

In order to enable this, you will need to have set your [local currency](advanced-settings-configuration.md#local-currency) in the Advanced Settings of the Branch Dashboard:

![](../../../assets/media/images/bc4195b3-2254-4e28-8be5-9928bf66ce16)
