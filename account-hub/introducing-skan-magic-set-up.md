SKAN Magic Set Up is Branch's new way to simplify mobile measurement for iOS campaigns ✨

Instead of manually implementing a custom configuration, you can now configure SKAdNetwork (SKAN) with a Branch-recommended conversion set up to help you get up and running quickly, while still maximizing campaign performance.

![](/img/0bc0676-7c07dc6-Screenshot_2023-10-06_at_2.02.55_PM.png)

This option is recommended for those brand new to SKAN, who want to prioritize user privacy while minimizing coding efforts.

![](/img/02610e5-8e4ecbf-Screenshot_2023-10-10_at_9.46.03_AM.png)

::: tip
Learn more about enabling SKAN and using Magic Set Up [here](enable-skadnetwork.md).
:::
