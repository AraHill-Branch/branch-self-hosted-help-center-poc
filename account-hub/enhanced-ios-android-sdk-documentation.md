Branch's most popular developer documentation pages have been updated! Integrating Branch into your mobile app has never been easier with these overhauled documentation updates 📰

## Overview

Recently, Branch published major documentation changes related to both the Android and iOS SDKs.

For both SDKs:

- The **Basic Integration** guides were simplified and updated for accuracy and clarity.
- The **Advanced Features** guides were given a complete refresh - they've been updated for usability, organization, and latest feature changes.
- The **Full Reference** guides were re-written from scratch. Now'll you'll find standardized documentation and code samples for every method made public by these two SDKs!
