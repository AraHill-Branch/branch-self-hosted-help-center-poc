Agency admins have the ability to add other users to their agency. To complete the steps below, be sure you're in the agency record, not the app.

## Add new agency users

::: info Agency users
Agency users are managed by agencies. Agencies themselves add members within their view. Make sure you are in your Agency account in Branch, not the client's Branch app account.
:::

1. In Branch, go to the [Account Settings](https://dashboard.branch.io/account-settings) page, and select the [Team](https://branch.dashboard.branch.io/account-settings/team) tab.
2. Select the **Add Agency Team Member** button.
3. Complete the following fields:

   - Email Address
   - First Name
   - Last Name
   - Select the **Access Level** you wish to grant the user, your options are:

     - **Agency Admin** - This grants full access to the Account Settings tab, including the ability to add, edit, and remove agency team members.
     - **Agency Team Member** - Read-only access on the Account Settings tab.

4. Select **Invite** when you're done.

   

::: info Disabled functionality
Any restricted functionality - either due to default Agency restrictions or selected access levels - in Branch is (1) grayed out, (2) unavailable, (3) accompanied by a pop-up modal informing the user about restricted access. See more about [default agency restrictions](add-agency-to-your-account.md#section-default-agency-access-level-restrictions).
:::

## Manage agency users

Agency admins can modify, resend invitations to, and delete agency team members from the following sections.

### Modify a team member's permissions

To modify an existing agency team member:

1. On the [Account Settings](https://branch.dashboard.branch.io/account-settings) page, on the [Agencies](https://dashboard.branch.io/account-settings/agencies) tab, find the team member you want to modify and select the `...` button in the **Actions** column for that user.
2. Select **Edit** to modify the agency team member.
3. Select **Save** when you're done.

::: danger Warning
To change the email address associated with an existing agency team member, contact [Support](https://support.branch.io/hc/en-us/requests/new).
:::

### Resend an invitation

To resend the invitation to join the account:

1. On the [Account Settings](https://branch.dashboard.branch.io/account-settings) page, on the [Agencies](https://dashboard.branch.io/account-settings/agencies) tab, find the team member you want to resend the invite to.
2. Select the `...`.
3. Select **Resend Invite**.

### Delete a team member

1. On the [Account Settings](https://branch.dashboard.branch.io/account-settings) page, on the [Agencies](https://dashboard.branch.io/account-settings/agencies) tab, find the team member you want to delete.
2. Select the `...`.
3. Select **Delete**.
4. In the **Are you sure you want to delete?** modal, select **Yes, Delete**.

### Request additional permissions

As a non-Admin user, you may not be able to access or use certain sections of Branch. If you'd like more access, contact your Agency Admin user on the account to ask for more permissions.
