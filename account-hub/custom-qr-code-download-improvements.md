Branch Custom QR Code customers now have access to higher-resolution custom Branch QR Codes, as well as the ability to download them in SVG format.

## Overview

Custom Branch QR Codes are getting a significant upgrade in the quality of the image that gets downloaded during creation.

They can also now be downloaded in three different file formats: PNG, JPEG, and SVG.

![](/img/a7f3499ef6cbe1006076073f1ddfd3a78cd1946cc20f703d24a960b804d386b0-Screenshot_2024-11-18_at_7.27.22_PM(1).png)

## Image Quality

The Branch QR Code image resolution is increasing to **300 pixels per inch** and the image size is increasing to **1667 x 1667 pixels**.

This improves the reliability of the Branch QR Code when it comes to scanning it via device cameras, whether presented digitally or in printed materials.

## SVG File Format

By popular request, Branch has introduced SVG files as an option for Branch QR Codes.

SVG (which stands for Scalable Vector Graphics) files are best used when being able to resize the image without losing quality is essential. This could include high-quality print materials (posters, flyers, large banners) or certain websites.
