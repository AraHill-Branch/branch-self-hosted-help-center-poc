## Overview

Branch is transitioning all self-attributing network (SAN) cost data to ROI Hub. This change affects cost data from Facebook, Google Ads, Snap, TikTok, and Apple Ads. Cost data from these SANs will no longer be automatically imported and displayed in analytics and exports through legacy systems.

Additionally, Branch’s existing Cost on Links solution is being deprecated in favor of ROI Hub. These changes align with Branch's goal of streamlining our products while providing customers with unified cost data and revenue insights, an improved user experience with intuitive design, and better visibility into ad partner connections.

## SAN cost data moving to ROI Hub

**Current behavior**: Cost data from SANs is automatically imported when these networks are authenticated, then displayed in analytics and exports.

**Upcoming change**: SAN cost data will only be available through [ROI Hub](roi-hub.md) as part of [Performance Pro](https://help.branch.io/v1/docs/packaging#performance-pro-tier). Legacy customers will need to purchase ROI Hub to continue accessing this data.

#### Affected ad networks

- Facebook
- Google Ads
- Snap
- TikTok
- Apple Ads

## Cost on Links to be discontinued

**Current behavior:** The Cost on Links solution allows customers to configure cost data via parameters on Branch Links, with ad networks populating cost values that Branch aggregates.

**Upcoming change:** The Cost on Links solution will be deprecated and also replaced by ROI Hub.

## More information

These changes support Branch’s goal of consolidating our infrastructure and making valuable insights easy for customers to find.

They also come with all the additional benefits of ROI Hub, which include:

- **Increased value:** ROI Hub unifies your cost data and revenue insights, and helps you understand the actual impact of your ad dollars.
- **Improved user experience:** The Cost on Links feature has complex implementation requirements, whereas ROI Hub boasts an intuitive, modern design.
- **Better visibility:** ROI Hub makes it easy to monitor the status of each of your ad partner connections, so you can find and fix issues quickly.

You can find **ROI Hub** in the Branch Dashboard under **Ads**, or access it directly [here](https://monster-factory-web.app.link/e/release-note-to-roi-hub-april-2025).

For questions or concerns about these changes, please reach out to your Branch point of contact.
