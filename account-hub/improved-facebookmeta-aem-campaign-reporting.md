Meta's Aggregate Event Measurement (AEM) campaigns will get new reporting updates in Branch. Get the details here 👍

## Overview

Starting July 2nd, Meta’s Aggregate Event Measurement ([AEM](https://www.facebook.com/business/help/721422165168355?id=1877298665783613)) attributed conversion events will be available in the Branch Dashboard and various data exports. This update offers customers additional insights into Meta attribution data, improving measurement and analysis of their performance.

This update will not affect other attribution sources or channels within Branch, which ensures existing processes and data from other sources remain unchanged.

### Data Availability

- AEM attributed conversions will be represented throughout Branch Dashboard reporting.
- AEM attributed conversion events will be available in log-level reporting. This includes opens and downstream conversion events.

  - Installs attributed to AEM are not available in log-level reporting.
