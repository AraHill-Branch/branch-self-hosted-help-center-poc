We've added five new IP addresses for Branch Webhooks to include in your server allowlist ➕

## Overview

For Branch to continue to send events to your servers via [Webhooks](webhooks.md), add the following IP addresses to your allowlist:

- 52.8.85.213
- 54.153.26.149
- 54.153.41.111
- 54.176.133.14
- 54.193.10.161

These additional IP addresses have been created to improve webhook reliability and will go into effect on June 18th, 2024
