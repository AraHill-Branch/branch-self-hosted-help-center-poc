## **Overview**

With link folders, you can organize your Branch Links by campaign, product line, team, or any structure that matches your workflow. Create folders, add links to them, and associate templates to automatically apply consistent tagging and configuration across all links in that folder.

::: info Note
Short Links are currently the only supported link type for folders. Ad Links are not currently supported.
:::

## **How it works**

With link folders, you can:

- Create and manage folders to organize your links
- Create links directly within folders (new links automatically belong to their parent folder)
- Move links between folders using the link action menu or edit flow
- Associate templates with folders (all links created in a folder automatically inherit the folder's template settings)
- Existing Link Collections from legacy Branch are automatically migrated to link folders with their template associations preserved

::: info Note
Previously created Link Collections from the legacy Branch experience will now be viewable as folders with their template associations preserved.
:::
