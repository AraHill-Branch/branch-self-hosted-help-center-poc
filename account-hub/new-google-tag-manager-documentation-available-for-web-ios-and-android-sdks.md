New user guides are here to simplify setting up Google Tag Manager to work with your Branch SDK! 💡

## Overview

Google Tag Manager (GTM) allows you to trigger specific actions related to Branch.

To do this, GTM calls various Branch functions included in your SDK integration.

Which functions are called and when depends on how you configure your tags and triggers in GTM.

## Implement Branch Features

Branch is happy to share three new guides that outline the steps you need to take in GTM and in your Branch code to trigger Branch features via GTM.
