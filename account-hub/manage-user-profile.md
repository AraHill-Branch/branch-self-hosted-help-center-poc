## Overview

::: danger Warning
The User tab is part of the [legacy Branch](branch-dashboard-overview.md) experience and is slated for deprecation.  
  
In the [new version](new-branch-overview.md) of Branch:

- Your first name, last name, email, avatar, and user ID live on the [Profile page](profile-new.md).
- Your Access Token lives on the Security and Access page, in the [Credentials tab](credentials-new.md).

To learn about the new Branch experience and the features rolling out in 2026, visit our [New Branch Overview guide](new-branch-overview.md).
:::

On the [Account Settings](https://dashboard.branch.io/account-settings) section, on the [User](https://dashboard.branch.io/account-settings/user) tab, all users can access the following:

- **Link with your Github account** if you wish to login with your Github account.
- **Dashboard UID** - This assigned by Branch and cannot be changed.
- **Generate Access Token** - This token is used to extract data from Branch platform via various export API's.
- **First and Last Names**
- **Email Address**
- **Update Password** if you do not have SSO enabled on your account.

::: danger Warning
If you are using the [legacy version of Branch](branch-dashboard-overview.md) and need to change the email address associated with your account, please contact [Support](https://support.branch.io/hc/en-us/requests/new).
:::
