# Create Quick Link

<ApiOperation operationId="createQuickLink" />

<div class="api-search-only">

Quick Links API createQuickLink Create Quick Link Creates a Branch Quick Link — a managed deep link that surfaces on the Branch Dashboard. When you include `qr_code_settings` in the request, both the link and its customized QR code are generated instantly and become visible in Branch. Set `type=2` to surface the link in the Dashboard. Customizing a QR code through this endpoint requires access to both the Quick Links API and the QR Code API. Quick Links API branch_key The Branch Key of the originating app, found in the [Settings](https://help.branch.io/using-branch/docs/profile-settings) tab of your Branch Dashboard. channel Deep link channel feature Deep link feature set campaign Campaign name stage tags Social media tags type Set to 2 in order to see Branch Deep Link URLs in the Branch Dashboard (must also set `$marketing_title`). Note that type=2 is also required to customize a QR code using `qr_code_settings`. data $marketing_title data.$marketing_title Set the marketing title to see the URL in the Dashboard (must also set `type` to 2). $fallback_url data.$fallback_url Change the redirect endpoint for all platforms - so you don't have to enable it by platform. Note that Branch will forward all robots to this URL, which overrides any OG tags entered in the link. System-wide Default URL (set in Link Settings) $fallback_url_xx data.$fallback_url_xx Change the redirect endpoint for all platforms based on a lower-case Alpha-2 country code. $desktop_url data.$desktop_url Redirect URL for desktop devices - mobile users will default to the app store. $ios_url data.$ios_url Change the redirect endpoint for iOS App Store page for your app (set in Link Settings) $ios_url_xx data.$ios_url_xx Change the redirect endpoint for iOS based on a lower-case Alpha-2 country code. For example, $ios_url_de="..." would redirect Germany deep link clicks. $ipad_url data.$ipad_url Change the redirect endpoint for iPads. Falls back to the $ios_url value when unset. $android_url data.$android_url Change the redirect endpoint for Android Play Store page for your app (set in Link Settings) $android_url_xx data.$android_url_xx Change the redirect endpoint for Android based on a lower-case Alpha-2 country code. For example, $android_url_de="..." would redirect Germany deep link clicks. $samsung_url data.$samsung_url Redirect to Samsung Galaxy Store on Samsung devices. Only link level control. For example, instead of a random string of characters/integers, you can set the vanity alias as .app.link/devonaustin. duration In seconds. Only set this key if you want to override the match duration for deep link matching. This is the time that Branch allows a click to remain outstanding and be eligible to be matched with a new app session. This is default set to 7200 (2 hours). analytics ~channel analytics.~channel Use channel to tag the route that your link reaches users.
