# Deep Linking API

<BranchCredentialsBar />

## Overview

The Deep Linking API creates and manages Branch links programmatically — generate links at scale and tag them by channel and campaign for analytics.

## Authentication

All requests require a Branch Key. See [the API authentication overview](/apidocs/#authentication) for how to retrieve it and apply it to your requests.

## Endpoint

```
POST https://api2.branch.io/v1/url
Content-Type: application/json
```
